# Template Angular Project

Init with :

- Tailwind
- Angular Material

# Tailwind

Possible to add font, add in tailwind.config.js :
```
theme: {
    fontFamily: {
      'wedding': ["Wedding"],
      "windsong": ["WindSong"],
      "librebaskerville": ["LibreBaskerville"],
    },
    extend: {},
  },
```
